# Nagrik

One assistant for every citizen.

**Nagrik** is a unified citizen-assistant platform designed to bridge the gap between regional citizens and digital government welfare structures. It enables individuals to conversationalise profile building, check scheme eligibility, file grievances, search templates of government forms, view policy news, and interact with the state apparatus using native Indian languages through text or voice.

---

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/) (App Router, JavaScript)
- **UI & Styling**: [React](https://react.dev/), [Tailwind CSS](https://tailwindcss.com/), [Lucide Icons](https://lucide.dev/)
- **State Management & Logic**: Native React Hooks

---

## Getting Started

Follow these steps to run the application locally:

### 1. Install dependencies
```bash
npm install
```

### 2. Start the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application dashboard.
